# Animated Software Developer Portfolio

A fully animated, responsive portfolio website with a creative, dark aesthetic. Built with HTML, CSS, and JavaScript.

## Features
- Animated gradient background and floating blobs
- Scroll-triggered reveal animations
- Responsive layout with mobile navigation
- Interactive hover and parallax effects

## Usage
Open index.html in a browser.

## Notes
All visuals are CSS-driven and use gradients as placeholders. Replace text content with your personal details.
